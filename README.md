# 🤖 AI Coach Bot

An intelligent AI-powered coaching assistant designed to guide users with personalized suggestions, learning support, and conversational insights using Generative AI.

# 🚀 Overview

AI Coach Bot is a conversational AI application that helps users with:

Learning guidance

Career advice

Problem-solving assistance

Personalized recommendations

It leverages modern Generative AI / LLMs to simulate a smart coaching experience through natural conversations.


# ✨ Features

💬 Conversational AI Interface – Real-time interaction with users

🧠 AI-Powered Responses – Context-aware intelligent replies

📚 Personalized Coaching – Guidance based on user queries

⚡ Fast & Scalable Backend

🎯 User-Friendly UI

🔐 Secure API Integration


# 🛠️ Tech Stack

Frontend

# ⚙️ Installation & Setup

1️⃣ Clone the Repository
git clone https://github.com/Prabhatrai7/AI-Coach-Bot.git
cd AI-Coach-Bot

2️⃣ Setup Backend
cd backend
npm install
npm start

3️⃣ Setup Frontend
cd frontend
npm install
npm start

4️⃣ Environment Variables
Create a .env file:
OPENAI_API_KEY=your_api_key_here

# ▶️ Usage

Open the app in your browser
Start chatting with the AI coach
Ask questions related to:
Career guidance
Learning paths

Receive intelligent responses in real-time
